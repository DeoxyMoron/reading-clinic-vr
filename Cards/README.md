# Cards
Development of VR Platform for Block-based visual programming


## Keyboard Controls

* Controller Position
  * W/S = Forward/backward
  * A/D = Left/right
  * Q/E = Up/down
* Controller Rotation
  * J/L = Yaw
  * I/K = Pitch
  * U/O = Roll
* Block Interaction
  * "1" = Spawn "Card" block
  * "5" = Flip Card
  * "-" = Decrease parameter
  * "=" = Increase parameter
  * Spacebar = Grab block 
* Misc
  * Esc = Reset scene
  * R = Reset turtle position
